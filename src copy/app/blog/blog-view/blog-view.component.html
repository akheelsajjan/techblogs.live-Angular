What is Nodejs?

NodeJS is runtime Javascript environment to run Javascript outside the browser.
It is Open-source, cross-platform & written in c c++ & Javascript.

NodeJS allows us to write Javascript on server-side. So we can use Javascript both
on frontend and backend, NodeJS makes "JavaScript everywhere".

There are many frameWorks written in Javascript. One of them is Express with which
we can built REST API'S
